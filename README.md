# T-Shirt Store E-Commerce Website
A responsive e-commerce website for selling custom t-shirts, built as part of the Vibe Coding 101
course.
## Features
- Product catalog with 6+ t-shirt designs
- Shopping cart functionality
- Responsive design for mobile and desktop
- Checkout process with customer information form
- Admin panel for changing products
## Technologies Used
- HTML5 for structure
- CSS3 for styling and responsive design
- JavaScript for interactivity
- Replit for development environment
## Live Demo
[View the live site here](https://jesse-t-shirt-time-jessespears3.replit.app/)
## What I Learned
- How to create responsive web layouts
- JavaScript for interactive shopping cart
- CSS Grid and Flexbox for product displays
- Form handling and validation
